### SOURCE : Notes de session Marc — 22 mars 2026 — fiabilité : observation directe Marc

#### Le cycle — 3 600 ans ou 3 666 ans ?

Deux valeurs circulent dans les sources :
- **3 600 ans** : valeur Sitchin, dérivée des textes sumériens (shar = 3 600)
- **3 666 ans** : valeur alternative — certains chercheurs l'associent à une dérive progressive ou à une lecture différente des sources

**Calcul comparatif depuis 2026 :**

| Cycle | 3 600 ans | 3 666 ans |
|---|---|---|
| N (précédent) | ~1 574 av. J.-C. | ~1 640 av. J.-C. |
| N-1 | ~5 174 av. J.-C. | ~5 306 av. J.-C. |
| N-2 | ~8 774 av. J.-C. | ~8 972 av. J.-C. |

#### Calibration sur Thera/Santorin — cycle N

L'éruption de Thera/Santorin est datée par les carottes glaciaires du Groenland à **~1 628 av. J.-C.**

- Avec cycle 3 666 ans : N arrive à ~1 640 av. J.-C. → **écart de 12 ans** — convergence remarquable
- Avec cycle 3 600 ans : N arrive à ~1 574 av. J.-C. → écart de 54 ans

**Conclusion provisoire Marc :** le cycle 3 666 ans calibre mieux sur Thera que le cycle 3 600 ans.

#### Calibration sur l'Exode — cycle N

**Hypothèse de travail :** les plaies d'Égypte sont les effets du passage précédent de Nibiru (reconnue explicitement par AM — voir section SOURCE 1 ci-dessous).

**Datations de l'Exode selon les sources :**

| Hypothèse | Date | Commentaire |
|---|---|---|
| Biblique haute (1 Rois 6:1) | ~1 446 av. J.-C. | Thoutmosis III / Amenhotep II |
| Consensus académique | ~1 250-1 200 av. J.-C. | Ramsès II / Mérenptah |
| Expulsion des Hyksos | ~1 550 av. J.-C. | Ahmosis Ier |
| Convergence avec Thera (3 666 ans depuis 2026) | ~1 640 av. J.-C. | Fenêtre éruption Thera |

**Note Marc :** La datation exacte de l'Exode est secondaire. Ce qui compte est la fenêtre : **1 200 - 1 650 av. J.-C.** — tous les candidats sérieux tombent dans cette fourchette, qui correspond au cycle N de Nibiru quel que soit le cycle retenu (3 600 ou 3 666 ans).

#### Calibration sur le Déluge — cycle N-2

**Selon le tableau chronologique Marc et la fiche [[DÉLUGE - Analyse]] :**

- Déluge généralisé / fin de l'Atlantide : **~9 000 av. J.-C.** (tableau Marc)
- Cycle N-2 avec 3 600 ans : ~8 774 av. J.-C. → écart ~226 ans
- Cycle N-2 avec 3 666 ans : ~8 972 av. J.-C. → écart ~28 ans

**Convergence remarquable avec 3 666 ans.** La fin du Younger Dryas est datée par les carottes glaciaires à **~9 700 av. J.-C.** — trop ancien pour N-2 des deux cycles. En revanche, Ryan & Pitman datent la rupture du Bosphore à **~5 600 av. J.-C.** → proche du cycle N-1 (3 666 ans : ~5 306 av. J.-C.).

❓ **Question ouverte :** Le cycle N-2 (~8 972 av. J.-C.) correspond-il au déluge généralisé/fin de l'Atlantide, ou à un autre événement (pole-shift, Younger Dryas) ? Le Younger Dryas (~9 700 av. J.-C.) est trop ancien pour le cycle N-2 — ou le cycle est plus long que 3 666 ans pour ce passage ?

#### La distinction fondamentale établie par Marc

**Effets directs de Nibiru** (gravitationnels, magnétiques, débris) ≠ **Actes délibérés anunnaki** (destruction barrage Bosphore, guerres nucléaires ~-2000, etc.)

Les fiches du dossier mélangent souvent les deux. La chronologie ADAPA doit distinguer :
- Effets Nibiru : séismes, pole-shift, tsunamis, volcanisme, météorites, perturbations magnétiques
- Actes anunnaki : décisions de destruction ou de préservation *pendant ou entre* les passages

**Exemple clé :** La rupture du Bosphore (~5 400 av. J.-C.) = acte anunnaki délibéré (Ninurta), pas effet direct de Nibiru. Lefebvre documente que les Igigis ont entretenu le barrage pendant ~3 600 ans avant de se rebeller — soit précisément un cycle de Nibiru. Lien de causalité entre passage N-1 et rébellion des Igigis : à documenter.
