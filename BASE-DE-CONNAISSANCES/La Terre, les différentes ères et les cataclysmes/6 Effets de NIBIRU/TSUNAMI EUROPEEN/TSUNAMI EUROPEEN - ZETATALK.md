## 3. Mécanisme du tsunami

**Source :** Page 1 — European Tsunami

Mécanisme **différent d'un tsunami classique** :
- Tsunami classique : plaque monte ou descend → onde directionnelle
- Tsunami européen : la dorsale atlantique **s'écarte** — rift qui s'ouvre
- L'eau s'engouffre dans le vide → remonte au centre du rift → sans direction privilégiée
- Devient **fortement soumise au Gulf Stream**

**Rôle du Gulf Stream :**
- Portion qui se recourbe → retourne vers le rift → retombe sans impact Afrique/Caraïbes
- Portion qui file vers le nord → **assault directement les côtes britanniques**
- Secondairement : France atlantique, Norvège
