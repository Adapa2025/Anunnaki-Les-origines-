## 🌐 THÈMES TRANSVERSAUX — Résumé

### 1. Mort & Résurrection du dieu
Baal (Canaan) / Osiris (Égypte) / Dumuzi (Mésopotamie) / Christ (Chrétien)
→ Mythe saisonnier universel. Baal = prototype le plus ancien documenté.

### 2. Dieu de l'orage vs Chaos marin
Marduk/Tiamat — Baal/Yam — YHWH/Léviathan — Zeus/Typhon
→ [[BAAL WIKI-NOTES]] / [[MOLOCH WIKI-NOTES]]
→ Lotan cananéen = Léviathan hébreu (étymologie identique)

### 3. Sacrifice des enfants / du premier-né
Moloch/Baal Hammon (Tophet) — Tlaloc — Kronos — Abraham/Isaac
→ [[MOLOCH WIKI-NOTES]]
→ Tophet = Géhenne (enfer chrétien) : même lieu géographique

### 4. Déluge / Reset de l'humanité
Ziusudra/Utnapishtim (Mésopotamie) = Noé (Hébreu) = Deucalion (Grèce) = Manu (Inde)
→ Récit unique diffusé depuis Mésopotamie

### 5. Descente aux Enfers / Katabase
Inanna (-2100 av. J.-C.) → Baal/Mot → Perséphone → Orphée → Christ aux Limbes
→ Inanna = première descente documentée

### 6. Dieu-Serpent / Sagesse cachée
Enki/Ningishzidda (caducée) → Serpent Éden → Apophis → Quetzalcóatl → Nāga
→ Nachash hébreu = 'observateur/devin' (pas nécessairement serpent animal)

### 7. Monothéisme comme usurpation de Marduk *(interprétation Lefebvre)*
Vers -2000 av. J.-C. : Marduk instaure la suprématie d'un dieu unique, efface les déesses.
YHWH = aboutissement fonctionnel de cette trajectoire.
*(NON consensus académique — position Lefebvre, à distinguer)*

### 8. Démonisation des dieux vaincus
Baal → Béelzébuth / Astarté → Astaroth / Moloch → démon / dieux greco-romains → démons chrétiens
→ LES DEMONS dans vault = panthéon antérieur démonisé de façon systématique
