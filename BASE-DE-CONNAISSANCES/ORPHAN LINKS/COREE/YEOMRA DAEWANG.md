# YEOMRA DAEWANG

Yeomra Daewang (염라대왕, 閻羅大王) — Grand Roi Yeomra, souverain de [[JEOSEUNG WIKI-NOTES|Jeoseung]] (monde des morts coréen). Juge suprême des âmes des défunts. Équivalent coréen du Yama bouddhiste/hindou, via le *Yanluo* (閻羅) chinois. Préside le tribunal central parmi les dix tribunaux des Sipwang. Attesté dans les rituels chamaniques (*gut*) et dans les textes bouddhistes coréens médiévaux.
