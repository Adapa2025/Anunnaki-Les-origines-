# NOAIDI

Chamane sámi — médiateur entre le monde des vivants, le monde des dieux (Ailikas) et le monde souterrain (Saivoaimo, Rotaimo, Jabmiidáibmu). Instruments : tambour magique (*meavrresgárri/goavddis*) peint de symboles cosmologiques, marteau en bois de renne. Pratique le joik (chant chamanique) pour entrer en transe. Voyage extatique en forme animale (renne, ours, loup, renard, zibeline). Fonctions : guérir les malades, prédire l'avenir, favoriser la chasse, combattre les chamanes adverses. Les Scandinaves médiévaux considéraient les Sámi comme de grands magiciens — documenté dans *Historia Norwegiæ* (XIIIe s.). Connexion ADAPA : même archétype institutionnel que l'angakkuq inuit, la Machi mapuche, le N'anga tsonga, le Babalawo yoruba — interface chamanique entre l'humain et le cosmique.
