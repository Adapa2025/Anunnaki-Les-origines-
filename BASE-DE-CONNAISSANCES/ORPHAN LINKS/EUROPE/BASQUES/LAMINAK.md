# LAMINAK

Laminak (sg. lamina) — créatures féminines surnaturelles de la mythologie basque. Mi-femme mi-oiseau ou mi-poisson selon les variantes régionales. Vivent dans les cours d'eau, les grottes, les fontaines. Tisserandes nocturnes. Capturées ou épousées par des humains dans les contes — le mariage échoue toujours quand l'humain viole un interdit (ne pas les voir peigner leurs cheveux, etc.). Structure de conte type "épouse surnaturelle". Source : Barandiaran.
