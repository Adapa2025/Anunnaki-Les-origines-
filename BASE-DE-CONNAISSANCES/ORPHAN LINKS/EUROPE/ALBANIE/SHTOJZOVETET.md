# SHTOJZOVËTET

Shtojzovëtet — esprits féminins nocturnes albanais, distinctes des Zana. Associées à l'espace interstitiel entre les mondes, actives la nuit. Peuvent être bénéfiques ou maléfiques. Liées aux naissances dans certaines traditions — présentes lors de la naissance d'un enfant pour en déterminer le destin. Sources : folklore albanais, Tirta.
