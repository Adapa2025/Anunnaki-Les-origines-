# KANUN

Kanun — code de droit coutumier oral albanais, codifié par écrit sous le nom de Kanun de Lekë Dukagjini (XVe s.) pour le nord de l'Albanie. Régit les relations sociales, la vengeance (gjakmarrja), l'hospitalité (besa), les rites funéraires. Vecteur principal de transmission des croyances cosmologiques albanaises pré-chrétiennes. Maintenu vivant dans les régions montagnardes jusqu'au XXe s. malgré les interdictions successives (ottomane, communiste).
