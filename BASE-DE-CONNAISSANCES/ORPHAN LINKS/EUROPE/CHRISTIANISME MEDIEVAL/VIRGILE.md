# VIRGILE

Virgile (Publius Vergilius Maro, 70-19 av. J.-C.) — poète latin, auteur de l'*Énéide* (dont le chant VI décrit la descente d'Énée dans l'Averne). Guide de Dante dans l'Enfer et le Purgatoire — représente la Raison humaine, capable de mener jusqu'au seuil du Paradis mais pas au-delà (non baptisé, réside dans les Limbes). Sa descente aux Enfers dans l'*Énéide* est la source directe de la géographie infernale dantesque.
