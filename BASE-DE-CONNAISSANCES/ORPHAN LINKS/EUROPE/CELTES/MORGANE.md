# MORGANE

Morgane (Morgan le Fay, Morgain) — fée guérisseuse, gouvernante d'Avalon dans le cycle arthurien. Demi-sœur d'Arthur. Conduit Arthur mourant vers Avalon après la bataille de Camlann. Figure ambivalente dans les sources — bienveillante dans les versions les plus anciennes (Geoffroy de Monmouth), plus sombre dans les versions ultérieures (Malory). Sources : Geoffroy de Monmouth (*Vita Merlini*, vers 1150), cycle Lancelot-Graal.
