# TENGRI BACHKIR

Tengri — ciel divin dans la tradition bachkire, principe céleste suprême. Représenté comme la grande yourte céleste dont le trou de fumée est l'étoile polaire. Tradition turcique distincte du Tengri mongol par les détails mythologiques locaux. Documenté dans l'épopée Ural-Batyr et les collectes ethnographiques.
