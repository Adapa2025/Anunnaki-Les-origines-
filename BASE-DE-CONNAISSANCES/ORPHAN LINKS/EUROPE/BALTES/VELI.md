# VELI

Veli (letton : Veļi, sg. Velis) — esprits des morts dans la tradition lettone, ancêtres défunts séjournant dans le domaine de Velu māte. Reviennent rendre visite aux vivants lors du *Veļu laiks* (fin octobre-début novembre). Distinct des esprits errants — les Veli sont des ancêtres honorés en relation régulière avec les vivants.
