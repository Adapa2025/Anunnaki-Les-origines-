# TUONI

Tuoni (et Tuonetar, sa fille/épouse selon variantes) — dieu/déesse des morts finlandais, gouvernant de Tuonela. Reçoit les morts à leur arrivée. Ses enfants sont décrits comme monstrueux dans le Kalevala. Figure d'autorité dans le monde des morts, pas de dimension morale punitive. Sources : *Kalevala*, Runes XIV-XVI.
