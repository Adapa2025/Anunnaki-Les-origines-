# LINGO PEN

Lingo Pen (Pahandi Kupar Lingal) — héros culturel et dieu gond. Descend dans le monde souterrain pour récupérer les instruments de musique sacrés (*Gond Bana*) volés par les divinités souterraines. Structure de descente initiatique. Figure centrale de la tradition orale gond transmise par les bardes *Pardhans*. Sources : Verrier Elwin, Shamrao Hivale.
