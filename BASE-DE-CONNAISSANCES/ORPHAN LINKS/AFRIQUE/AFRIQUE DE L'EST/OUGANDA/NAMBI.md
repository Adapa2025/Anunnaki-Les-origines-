# NAMBI

Fille de Ggulu (roi de Cloud Land) dans la cosmologie Baganda. Découvre Kintu lors d'une descente sur Terre via l'arc-en-ciel. Décide de l'épouser malgré l'opposition initiale de son père. Épouse Kintu après les cinq épreuves. Cause involontaire de l'entrée de la mort sur Terre — remonte à Cloud Land chercher le mil oublié pour sa poule, malgré l'interdiction de revenir, permettant à son frère Walumbe de suivre le couple. Mère des premiers enfants Baganda. *Nambi* signifie "celle qui a faim" ou "celle qui mange" — référence au mil oublié.
