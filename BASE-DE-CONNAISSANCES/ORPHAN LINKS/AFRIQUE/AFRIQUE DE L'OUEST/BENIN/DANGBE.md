# DANGBE

Vodun du serpent python dans la cosmologie Fon. Esprit du python royal (*Python regius*), animal sacré dont le meurtre était passible de mort dans le royaume du Dahomey. Centre de culte principal : Ouidah (Bénin), ville où un temple lui est encore consacré. Symbole de fertilité, de prospérité et de protection royale. Correspond au Dan haïtien (Danbala Wedo).
