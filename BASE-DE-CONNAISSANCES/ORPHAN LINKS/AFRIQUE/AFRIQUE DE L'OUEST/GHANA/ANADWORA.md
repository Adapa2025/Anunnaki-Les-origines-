# ANADWORA

Les huit Seigneurs de la nuit (*Anadwora*) qui gouvernent Asamando dans la cosmologie Akan. Chacun représente un type de mort ou de souffrance : Basashadie (chef, le Impitoyable — manteau de sauterelles), Yareɛra (maladie), Yirira (noyade), Wudinera (meurtre), Dadara (vieillesse), Hweasera (chutes), Ɛkɔmra (famine), Awudurobonɛra (poison/mauvaise médecine). Analogue fonctionnel aux Seigneurs de Xibalba (Maya) et aux neuf niveaux de Mictlan (Aztèque).
