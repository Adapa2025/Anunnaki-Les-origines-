# JENGI

Esprit suprême de la forêt dans la cosmologie Baka. Aussi appelé *Djengui* ou *Ejengi*. Médiateur entre Komba (dieu créateur) et les Baka. Omnipresent dans la forêt — punit les transgresseurs dans ses frontières. Se manifeste uniquement quand l'harmonie règne dans la communauté. Apparaît lors des rituels comme une créature enveloppée de raphia roux, sans visage ni membres visibles. Central dans l'initiation masculine (rite Jengi) — les initiés obtiennent le droit de circuler librement dans la forêt sacrée. Vénéré après la chasse lors du rituel Luma.
