# BANKAMBO

Ancêtres dans la cosmologie Luba (sing. *mukambo*). Résidents du Village des Ancêtres, accessible après une vie éthique conforme aux valeurs ancestrales. Généralement bienveillants — protègent leurs descendants, annoncent les grossesses dans les rêves, conseillent via la divination (*lubuko*). Certains deviennent malveillants après la mort — la société secrète Tusanji était chargée de neutraliser ces esprits hostiles. Consultés par les devins Bilumbu en état de possession.
