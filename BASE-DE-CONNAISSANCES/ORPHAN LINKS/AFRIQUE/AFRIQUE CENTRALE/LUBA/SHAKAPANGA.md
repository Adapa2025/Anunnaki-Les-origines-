# SHAKAPANGA

Nom principal du dieu créateur suprême dans la cosmologie Luba. Aussi désigné par : Leza, Vidye, Efile Mukulu, Maweja, Mulopo. Omniprésent — *"Dieu n'est en aucun lieu, Dieu est partout"*. Pas de sanctuaire dédié. Sa présence se manifeste dans tout lieu de puissance (arbres puissants, cascades, collines). La trickster Kafilefile est son antagoniste principal. Analogue de Ngewo (Mende), Roog (Serer), Nyame (Akan) — créateur transcendant sans culte direct.
