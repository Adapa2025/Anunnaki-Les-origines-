# AMA-NO-IWATO

Ama-no-Iwato (天岩戸), "Grotte Rocheuse Céleste" — grotte dans [[TAKAMAGAHARA WIKI-NOTES|Takamagahara]] où Amaterasu se retire après la visite perturbatrice de Susanoo. Son retrait plonge le monde dans l'obscurité totale. Les huit millions de kami (*yaoyorozu no kami*) réunis devant la grotte organisent des festivités bruyantes pour l'attirer dehors — Amaterasu, curieuse des rires, entrouvre la porte et est saisie par Ame-no-Tajikaraō. Le retour d'Amaterasu rétablit la lumière dans le monde. Sources : *Kojiki* (712), *Nihon Shoki* (720).
