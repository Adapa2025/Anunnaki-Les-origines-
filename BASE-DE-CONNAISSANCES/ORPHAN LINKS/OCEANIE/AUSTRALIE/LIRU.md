# LIRU

Serpent venimeux mâle — ancêtre du Tjukurpa Anangu. Vient du sud-ouest, s'installe près de Mutitjulu, tue le neveu de Kuniya. Affronte Kuniya dans le combat qui marque la fin du Temps du Rêve. Ses deux entailles de lance sont visibles dans la paroi gauche de la gorge de Mutitjulu. Trace rituelle vivante : les Anangu qui doivent se faire pardonner quelque chose d'important se font deux entailles sur la tête — comme Liru.
