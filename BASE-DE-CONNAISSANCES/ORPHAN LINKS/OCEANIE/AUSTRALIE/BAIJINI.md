# BAIJINI

Peuple mystérieux mentionné dans le cycle de chants Djang'kawu des Yolŋgu (nord-est Arnhem Land). Les Yolŋgu sont "catégoriques" : les Baijini précédaient les Macassans. Description transmise : technologie avancée (métiers à tisser, agriculture), construisaient des huttes lors de leurs séjours, peau claire. Venaient collecter le trépang comme les Macassans. Intégrés dans le registre mythologique Wangarr (Temps du Rêve) — non dans la mémoire historique ordinaire, ce qui suggère une antiquité perçue comme très grande. Étymologie du mot débattue (Garry Trompf : dérivé du makassais "femmes"). Première synthèse académique : Peter Worsley (1955). Aucune preuve archéologique matérielle d'une présence distincte des Macassans à ce jour. Connexion ADAPA : population à technologie avancée et peau claire, d'origine inconnue, précédant les contacts historiques documentés — même structure que les Apkallu ou les Baijini dans d'autres traditions.
