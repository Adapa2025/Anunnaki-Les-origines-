# IDJAIR

Premier Wandjina — réside dans la Voie lactée. Père de tous les Wandjina. Tradition de Mowanjum (Derby, Kimberley). Son premier fils Wandjina Wallungunder descend sur Terre et joue le rôle central dans la création. Structure : Idjair (Voie lactée) → Wandjina Wallungunder (descend sur Terre) → Gyorn Gyorn (premiers humains) → peuples du Kimberley. Connexion structurelle : même hiérarchie que Anu (ciel) → Enlil/Enki (descendent sur Terre) → Lulu (humanité créée) dans le corpus sumérien.
