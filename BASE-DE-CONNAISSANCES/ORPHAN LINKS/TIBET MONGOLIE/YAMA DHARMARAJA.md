# YAMA DHARMARAJA

Yama Dharmaraja ("Roi du Dharma Yama") — juge des morts dans le Sidpa Bardo (troisième phase du Bardo tibétain). Pèse les actes noirs et blancs du défunt dans une balance. Ses assistants tiennent le miroir des actes. Figure issue du Yama védique/hindou, réinterprétée dans le cadre bouddhiste tibétain. Assimilé à Erleg Khaan dans la tradition mongole.
