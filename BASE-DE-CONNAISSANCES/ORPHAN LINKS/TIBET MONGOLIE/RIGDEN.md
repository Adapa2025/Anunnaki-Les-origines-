# RIGDEN

Rigden (tibétain : rigs ldan, "détenteur du lignage") — rois-prêtres de Shambhala, gardiens du Kalachakra. La liste comprend trente-deux Rigden. Le dernier, Rigden Dragpo ("Rigden le Féroce"), conduira les forces de Shambhala lors d'une bataille finale contre les forces du mal. Sources : Kalachakra Tantra, guides vers Shambhala (Panchen Lama XVIIIe s.).
