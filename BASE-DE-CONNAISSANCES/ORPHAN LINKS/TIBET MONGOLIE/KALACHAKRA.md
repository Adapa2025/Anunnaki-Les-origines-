# KALACHAKRA

Kalachakra (Roue du Temps, sanskrit : कालचक्र) — système tantrique bouddhiste majeur du vajrayana tibétain. Ses enseignements seraient originaires de Shambhala — reçus par le roi Suchandra directement du Bouddha historique. Le Laghu Kalachakra Tantra (Xe-XIe s.) est le texte fondateur. Les initiations Kalachakra données par le Dalaï-Lama sont parmi les plus grandes cérémonies bouddhistes actuelles.
