# SIRAT

Sirat (الصراط, al-Sirat al-Mustaqim dans le Coran = "le droit chemin") — dans l'eschatologie islamique, pont tendu au-dessus de Jahannam que traversent les âmes après le Jugement Dernier. Décrit dans les Hadiths comme plus fin qu'un cheveu et plus tranchant qu'une épée. Les croyants le traversent à des vitesses variables selon leurs mérites (certains comme l'éclair, d'autres en rampant) ; les mécréants tombent dans Jahannam. Parallèle structurel direct avec le Pont Chinvat zoroastrien et le Pont Chinvat.
