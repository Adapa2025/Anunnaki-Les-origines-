# TABLETTES NAACAL

Prétendues tablettes d'argile gravées dans le "dialecte sacré de Mu" — source principale invoquée par James Churchward pour étayer l'existence du continent Mu. Selon Churchward, ces tablettes lui auraient été enseignées par un "vieux sage indien" dans un temple de l'Inde britannique. Dans un autre livre, les mêmes tablettes se trouvent au Tibet. Aucune tablette Naacal n'a jamais été présentée à un chercheur tiers, photographiée, transcrite indépendamment, ni localisée dans quelque institution ou collection. Leur existence repose entièrement sur l'affirmation de Churchward, sans corroboration d'aucune sorte. Le terme "Naacal" lui-même n'apparaît dans aucun texte indien ancien connu. Cas classique de source invérifiable servant de fondation à une théorie entière.
