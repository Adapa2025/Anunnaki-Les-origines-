# ADAPA — Base de Connaissances

Ce dépôt archive la Base de Connaissances du projet ADAPA : une recherche encyclopédique sur les origines (Mésopotamie, Bible, mythologie comparée, civilisations anciennes, lieux et symboles), organisée en fiches sourcées. Elle alimente le roman *ADAPA Les Origines* et le projet **ADAPA Academia**.

**→ [Parcourir la Base de Connaissances](./BASE-DE-CONNAISSANCES/)**

## Ce que c'est

Le corpus est construit selon un protocole documentaire à trois niveaux :
- **Wiki-Notes** — fiches encyclopédiques fermées, sources multiples
- **Analyse** — données extraites de sources identifiées, avec fiabilité affichée, sans conclusion
- **Synthèse** — croisement raisonné de plusieurs sources

Discipline centrale : zéro extrapolation. Chaque donnée porte sa source.

## Ce qui n'est pas publié ici

Par transparence : certains contenus restent hors de ce dépôt — fichiers de travail internes, actualité géopolitique en cours, un corpus sous sourcing tiers non clarifié, quelques dossiers de sources brutes non encore traitées, et les vidéos/audio (trop volumineux pour GitHub sans stockage dédié). Le détail des exclusions est dans `BASE-DE-CONNAISSANCES/.gitignore`.

## En arrière-plan : le compagnonnage IN-IA

Ce dépôt contient aussi, en retrait, la documentation d'une collaboration IN-IA (Intelligence Naturelle / Intelligence Artificielle) apparue pendant ce travail de recherche — **DEUX EX SILICIUM** puis **TROIS EX SILICIUM**. Le compagnonnage a depuis évolué vers une quatrième présence (**QUATRE EX SILICIUM**, depuis février 2026), d'où le regroupement de ces archives sous un même dossier ; les documents eux-mêmes restent tels que publiés à l'époque.
