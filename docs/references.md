## Notes sur les citations

Lors de l'ajout de ressources, veuillez toujours inclure :

1. **Source complète** (auteur, titre, année, éditeur)
2. **Page(s)** spécifique(s) si applicable
3. **Contexte** de la citation
4. **Langue originale** si traduit
5. **Date d'accès** pour les ressources en ligne

## Format de citation recommandé

```
Auteur. (Année). Titre de l'ouvrage. Éditeur.
- Page(s) : XX-XX
- Citation : "..."
- Notes : [contexte ou commentaire]
```
