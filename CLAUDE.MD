# Anunnaki - Les Origines

## Project Overview

This repository is a collection of resources concerning "Les Origines" (The Origins), focusing on the Anunnaki and ancient Mesopotamian mythology and history.

## Purpose

The project aims to gather, organize, and present research materials, documentation, and resources related to:
- The Anunnaki deities from ancient Mesopotamian cultures
- Origin stories and creation myths
- Historical and archaeological context
- Cultural and mythological significance

## Repository Structure

Currently a documentation and resource collection project. The structure may evolve to include:
- Research documents and papers
- Historical references
- Mythological texts and translations
- Visual resources (images, diagrams)
- Bibliography and citations

## Development Guidelines

### Content Organization
- Keep resources well-organized by category or theme
- Maintain clear documentation and references

### Language
- Primary language: French
- Include translations where applicable
- Maintain academic/research tone

### File Naming
- Use descriptive, clear file names
- Follow consistent naming conventions
- Use hyphens or underscores for spaces
